## ⚙️ Architecture & Local Setup

This project is fully containerized to ensure consistency across development and production environments. To run this project locally, you only need Docker installed.

1. Clone the repository:
   `git clone https://github.com/Byonix25/Portfolio.git`
2. Navigate to the directory:
   `cd Portfolio`
3. Spin up the containers:
   `docker-compose up --build`

The application will be available at `http://localhost:3000`.
